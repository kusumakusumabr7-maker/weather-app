# weather-app 🌦️

A Node.js-based weather forecasting application.  
It uses the OpenWeatherMap API to show real-time weather with dynamic visuals.

## Features
- Real-time weather updates
- Dynamic backgrounds and icons
- Smooth transitions for better user experience

## How to Run
1. Clone the repo
2. Run `npm install`
3. Start the server with `node server.js`
4. Open `http://localhost:3000` in your browser
